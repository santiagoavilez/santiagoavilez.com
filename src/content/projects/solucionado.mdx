## Introducción.

Hoy quiero compartir con ustedes un caso de éxito que destaca mi capacidad para liderar proyectos, coordinar equipos y utilizar las tecnologías más avanzadas en el desarrollo de soluciones web. Este proyecto se llama "Solucionado App", una plataforma que conecta prestadores de servicios domésticos con clientes potenciales.


<a href="https://solucionado.com.ar" target="_blank">Sitio web de Solucionado App!</a>

### Descripcion del proyecto

**Solucionado App** es un MVP (Minimum Viable Product) desarrollado para una startup tecnológica cuyo objetivo es simplificar la contratación de servicios domésticos. La plataforma permite a los usuarios encontrar y contratar profesionales de limpieza, reparación, y otros servicios de manera eficiente y segura.

> Responsabilidades y Rol

En este proyecto, asumí el rol de Desarrollador Web y Líder de Proyecto. Mis principales responsabilidades incluyeron:

- **Liderazgo de Equipo:** Dirigí un equipo de tres desarrolladores, asegurando una comunicación fluida y efectiva con el cliente para definir y priorizar los requisitos funcionales del proyecto.

- **Gestión de Requerimientos:** Hablé con el cliente, Francisco Maldonado, para gestionar sus requerimientos y convertirlos en features. Planifiqué y diseñé el plan de implementación para el equipo, priorizando las tareas según su importancia.

- **Reuniones de Feedback:** Coordiné reuniones regulares con el cliente para obtener feedback, corregir errores y mejorar la aplicación continuamente.

- **Selección del Stack Tecnológico:** Tomé la decisión de seleccionar un stack tecnológico moderno y emergente (Next.js, TypeScript, tRPC, TailwindCSS, ShadcnUI, Prisma) para desarrollar la aplicación de la forma más rápida y eficiente posible, asegurando que fuera funcional, escalable y atractiva, a pesar del riesgo de no conocer del todo bien las tecnologías ni su documentación.

> Tecnologías y Servicios Utilizados

Para asegurar un producto robusto y eficiente, implementé una variedad de tecnologías y servicios:

- **Autenticación: Clerk** para gestionar la autenticación de usuarios de manera segura.

- **Gateway de Pagos:** Mercado Pago para facilitar transacciones seguras y rápidas.
- **Almacenamiento y Optimización de Imágenes:** Cloudinary para manejar y optimizar las imágenes de manera eficiente.

- **Mensajería:** Twilio (WhatsApp) para permitir la comunicación directa entre clientes y prestadores de servicios.

- **Mailing Programable:** Mailersend para gestionar las comunicaciones por correo electrónico de manera automatizada.

### Logros y Resultados

El proyecto Solucionado App fue un éxito rotundo, con varios logros destacados:

- **Implementación Exitosa del MVP:** Logramos entregar un producto funcional con todas las características principales solicitadas por el cliente.

- **Mejora en la Eficiencia del Equipo:** Gracias a una asignación clara de tareas y objetivos, el equipo trabajó de manera más eficiente y coordinada.

- **Satisfacción del Cliente:** Francisco Maldonado quedó sumamente satisfecho con el resultado, destacando la calidad y la rapidez en la entrega del proyecto.

> Impacto y Beneficios

La implementación de Solucionado App no solo mejoró la experiencia de los usuarios al facilitar la contratación de servicios domésticos, sino que también permitira a la startup posicionarse como un competidor fuerte en el mercado. La plataforma escalable y bien diseñada asegura que la empresa pueda crecer y adaptarse a nuevas necesidades sin problemas técnicos significativos.
